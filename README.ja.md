# まちかちょうか

日本の市区町村の名称を「まち」か「ちょう」で正しく判別できるかを確認するクイズアプリです。

## デモ
[クイズ - まち or ちょう?](https://code4fukui.github.io/machiorcho/)

## 機能
- 日本の市区町村名を使ったクイズ問題
- 正解率の表示
- 関連情報へのリンク

## データ・API
- 問題データ: [日本の市区町村一覧CSV](https://code4fukui.github.io/address-japan/data/city.csv)
  - [アドレス・ベース・レジストリ by デジタル庁](https://www.digital.go.jp/policies/base_registry_address/)
  - [address-japan by Code for FUKUI](https://github.com/code4fukui/address-japan)
