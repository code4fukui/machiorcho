# machiorcho

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A quiz app that tests the user's knowledge of Japanese city and town names.

## Demo
[Quiz - 町 or 庁?](https://code4fukui.github.io/machiorcho/)

## Features
- Quiz questions based on Japanese city and town names
- Checks if the user can correctly identify if a name ends in "machi" (町) or "cho" (庁)
- Tracks the user's score and displays the final score

## Data / API
The quiz questions are generated using data from the [Address Base Registry by the Digital Agency](https://www.digital.go.jp/policies/base_registry_address/), provided by [Code for FUKUI](https://github.com/code4fukui/address-japan) in the [city.csv](https://code4fukui.github.io/address-japan/data/city.csv) file.
